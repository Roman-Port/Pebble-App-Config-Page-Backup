Color options
Choose the background, hour, minutes, and weather/steps/date colors for the watchface. (For Pebble and Pebble Steel, select from black & white colors only. Black is default for background color and white is default for text color.)
Weather options
Check to use current GPS location for weather conditions or manually enter weather location information below.
Enter location for non-GPS weather condition retrieval, e.g. 'San Diego CA US', or '92103 US'
Check to show today's lo and hi temperatures. Also temporarily displays last weather update time.
Check to display temperature in degrees Celsius. Uncheck to display in degrees Fahrenheit.
Select how often weather information is updated (minutes).
Display options
Select font width for hour digits - thin, regular, or bold. Relaunch watchface to apply display option.
Select font width for minutes digits - thin, regular, or bold. Relaunch watchface to apply display option.
