Alignment options
Select alignment for weather/date & time display lines.
Color options
Choose the background, time, and weather/date colors for the watchface. (For Pebble and Pebble Steel, select from black & white colors only. Black is default for background color and white is default for text color.)
Weather options
Check to use current GPS location for weather conditions or manually enter weather location information below.
Enter location for non-GPS weather condition retrieval, e.g. 'San Diego CA US', or '92103 US'
Check to enable shake option to show today's lo and hi temperatures. Also displays last weather update time on date line.
Check to display temperature in degrees Celsius. Uncheck to display in degrees Fahrenheit.
Select how often weather information is updated (minutes).
Display options
Check to display current date information.
Check to display "o" for single-digit minutes. For example, two o two or three o seven. Relaunch watchface to apply display option.
Check to display minute lines using a bold font. Relaunch watchface to apply display option.
Check to display weather & date lines using a larger font size. Relaunch watchface to apply display option.
Additional options
Check to vibrate on Bluetooth disconnect. Hour hand will also turn blue when disconnected.
