Weather options
Check to use current GPS location for weather conditions or manually enter weather location information below.
Enter location for non-GPS weather condition retrieval, e.g. 'San Diego CA US', or '92103 US'
Check to enable shake option to show today's lo and hi temperatures. Also temporarily displays last weather update time at bottom of screen.
Check to display temperature in degrees Celsius. Uncheck to display in degrees Fahrenheit.
Select how often weather information is updated (minutes).
Display options
Select to invert black & white colors.
Select to blink the colon between the hours and minutes with every second.
Check to vibrate on the hour.
Check to display weather text items using a larger font size.
Additional options
Check to vibrate on Bluetooth disconnect.
Note: Some display elements require that the watchface is relaunched to properly update alignment.
