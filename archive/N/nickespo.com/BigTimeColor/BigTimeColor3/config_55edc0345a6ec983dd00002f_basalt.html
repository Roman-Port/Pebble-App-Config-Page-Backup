  • Release Notes

    •   4.5 (Nov. 14, 2016) Now supports the new Pebble Time 2 (which has an increased pixel density, and so allows larger/smoother digits).
        4.0 (Sept. 28, 2016) Support added for the new Pebble 2.
        3.3 (Dec. 28, 2015) As promised, those of you with B&W Pebble and Pebble Steels now have two font colors (white and gray), and can customize their time font colors! Also, fixed a very embarrassing calendar issue in which December had 30 days! (Thank you, Rachel Y.!)
        3.1 (Dec. 9, 2015) Added B&W (O.G. Pebble and Pebble Steel) support. No changes for Pebble Time.
        3.0 (Dec. 4, 2015) Major new release. Added ability to set multiple colors, and for no adequately-explored reason (other than it's winter here in the Northern Hemisphere) -- optional snow-covered digits (no, not falling snow, that'd kill the battery)! Stay tuned for more...
        2.5 (Oct. 31, 2015) Bug fix release, and modified some code to improve battery life.
        2.4 (Oct. 28, 2015) Added optional battery indicator, made Bluetooth indicator optional, and for Pebble Round, moved the battery and Bluetooth indicators to the center-left and center-right of the screen.
        2.3 (Oct. 17, 2015) REALLY fixed sporadic crashes due to shake/calendar functionality, and added Pebble Round compatibility.
        2.1 (Oct. 11, 2015) Fixed sporadic crashes due to shake/calendar functionality
        1.2 (Oct. 2, 2015) Fixed crash related to calendar month rollover
        1.0 (Sep. 7, 2015)First release
