Talk2Web Configuration

With this app you can send dictation result to your own server, you can use it for voice control your smart house with Pebble for example
Your server URL
Send dictation results to this url with GET parameter text
Dicatation module settings
Append Pebble Account Token to url with GET parameter token, can be used for auth
Append coordinates to url with GET parameters lat and lon (can eat your battery)
You can customize server URL much more with this, use text, token, lat, lon variables, example (it is JavaScript):
"http://yourserver.com/" + text.split(" ")[0] + "/?value1=" + text.substr(text.indexOf(" ") + 1) + "&value2=" + lat + "&value3=" + lon
Can be usefull with IFTTT integration
