Tempurature
This switches celsius to fahrenheit for the tempurature readout. Celsius is off and Fahrenheit is on.
Weather Source
This switch will tell the watchface how to use the location for weather information. On is using GPS and off is using Postal Code and Country.
Select Country and Postal Code
If not using GPS, you must select your country and enter your postal code to get a general weather data. Some phones may not allow GPS to work.
These settings only change on the watch after you click save.
